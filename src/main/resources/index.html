使い方

  • KdBもどきを使用する場合
    1. KdBもどきにPCでアクセスする
    2. 仮の履修時間割を組む
    3. 検索条件をクリアし、「お気に入り」にチェックを入れて検索する
    4. 画面最下部の「CSVダウンロード」をクリックする
    5. 移行要件ツールでダウンロードしたCSVファイルを選択する
    6. 「判定を開始」を押す
  • Twinsを使用する場合
    1. 実装中です……

・応募可能で重点科目上限単位数を超えている場合に緑、重点科目単位数を超えていない場合には黄、そのどちらでもない場合には赤で示しています。
・使用したCSVファイルはサーバーには保存されません。
・医療科学類の重点科目上限単位数、医学類の判定には対応しておりません。
